## 📁 About This Project
A fully responsive, SEO-optimized static travel agency website built with HTML5, CSS3, and JavaScript — deployed live on Netlify.

**Topic Selected:** Travel Website

## 📄 Pages
| File | Page |
|------|------|
| `index.html` | Home — hero banner, destinations, features, testimonials |
| `about.html` | About Us — company story, values, team |
| `services.html` | Tour Packages — Europe, Asia, and India packages |
| `gallery.html` | Photo Gallery — 12 destination images with lightbox |
| `contact.html` | Contact & Booking — enquiry form with validation + FAQ |

## 🛠 Technologies Used
- **HTML5** — semantic structure, accessibility attributes
- **CSS3** — custom properties, flexbox, grid, media queries
- **JavaScript (Vanilla)** — no frameworks or libraries

## ✨ JavaScript Features
1. Sticky navbar with scroll effect
2. Mobile hamburger menu toggle
3. FAQ accordion expand/collapse
4. Contact form validation with error messages
5. Gallery lightbox (click to enlarge + keyboard ESC close)
6. Scroll-reveal animations using IntersectionObserver

## 📱 Responsive Breakpoints
- Mobile: up to 768px
- Tablet: 769px – 1024px
- Desktop: 1025px+

## 🔍 SEO Implemented
- `<title>` tag on every page
- `<meta name="description">` on every page
- `<meta name="keywords">` on every page
- Single `<h1>` per page, `<h2>`/`<h3>` for sections
- `alt` text on every image
- Simple, descriptive file names
- Internal links between all pages

## 🚀 Deployment
This website is hosted on **Netlify** via GitHub integration.

### How to Deploy
1. Upload this folder to a public GitHub repository
2. Go to [netlify.com](https://www.netlify.com) → New Site → Import from Git
3. Connect your GitHub account and select this repository
4. Click **Deploy Site**
